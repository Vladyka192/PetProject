Simple Web UI — Pet Project (HTML + CSS)

Этот репозиторий — мой небольшой pet-проект, созданный для практики вёрстки, улучшения навыков работы с HTML и CSS, а также отработки принципов адаптивного интерфейса.

О проекте

Проект простой. Цель — показать:

работу с HTML 

простой, читаемый и поддерживаемый CSS

использование flexbox и grid

создание вёрстки с помощью двух языков


Функциональность

Собственный дизайн — аккуратная типографика, кастомные цвета, простые анимации

Чистая структура — разделение на логичные блоки, минимальный, структурированный CSS

Быстрая загрузка — без библиотек и лишнего кода

Стек технологий

HTML5

CSS3 (Flexbox / Grid, кастомные анимации, переменные)
